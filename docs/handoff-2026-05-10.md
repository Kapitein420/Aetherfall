## New features (player-facing)

### Title splash + flow
- Cinematic full-bleed splash on load. "PRESS ANY KEY" prompt with breath animation.
- Any key/click → 480ms fade to setup screen. Then play through encounter → run flow as before.

### Deck viewer
- "Preview deck" button on setup (one per player) → shows that class's 15-card starter grouped by role.
- "Deck (N)" button on each in-combat champion baseplate → shows the player's live deck with a `5 in hand · 10 in draw · 0 in discard` subtitle.
- Esc or × button to close.

### Rogue-lite run loop
- **Setup picker** now has a **Run length** row: 1× / 3× / 5× / 7×. Default 3×.
- Run-mode auto-generates an encounter list starting with the picked encounter, then random non-repeats.
- After each fight victory: **Reward screen** with crystals earned + 3 random card picks per player (skip allowed).
- **Boss encounter clears** (single-monster encounters) drop a 2-option relic pick.
- HP **persists between fights** — no auto-heal. Greenheart Sigil relic adds the heal back.
- Final encounter cleared → **Run-complete** splash with stats.
- All players fall → **Run-failed** splash, red-themed, "Try again".

### 8 starter relics
| Relic | Effect |
| --- | --- |
| Bulwark Heart | +5 max HP all players |
| Echo Stone | +1 card drawn on round 1 of each fight |
| Greenheart Sigil | Heal 5 HP per player at fight start |
| Stormrunner Anchor | +1 faction-matched token per fight |
| Fixate Lens | +2 damage on fixated targets |
| Networked Inverter | First card per round costs −1 |
| Glass Reactor | −8 max HP, +1 starting energy |
| Crystal Refractor | +5 bonus crystals per clear |

Action bar shows current run status: `Fight 2/3 · ◈ 5 · ❤ ✦` (fight number, crystals, owned relic icons).

### Two new playable classes (full 4-class roster)

**Bloomcaller** — Bio-Growth third canonical archetype
- HP 38, Sustain role, element `bio-growth`
- Deck: 4 Basic Attack · 4 Basic Defend · 4 Mend · 3 Greenbloom
- Loop: Greenbloom seeds tokens → passive boosts every Mend → spend tokens on basics when you need to attack.

**Stormtide Conduit** — Storm + Hydroflow hybrid
- HP 36, Striker role, element `storm-charge`
- Deck: 6 Basic Attack · 6 Basic Defend · 2 Dual Channel · 1 Storm Tide
- Twist: Basics spend *opposing* token types — Attack costs Storm, Defend costs Hydroflow. Storm Tide naturally fires the engine's existing **Conductive Surge** combo.

### Polish
- Stat-pill kit unified across player + monster baseplates (HP, Energy, Block, Defense, all same shape).
- Lighter panel chrome — 6% borders, single-stop gradients, dropped heavy drop-shadows.
- Per-monster intent chip — damage tier color (light/medium/heavy/critical), target dot class-tinted, block forecast badge.
- Token spend preview pulse — when armed card has a `spendToken` rider, matching chip pulses with a `−1` badge.
- Card hover zoom — hovered cards scale 1.4× upward so rules text is readable without queueing.
- Round summary toast — 5-second post-resolve recap of damage taken + dealt.
- Animation calm-down — every infinite loop slowed to 2.6–3.4s breath, no horizontal shake on HP damage, no overlapping fast tempos.
