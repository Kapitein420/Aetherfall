# Card Assets

The browser prototype looks for generated full-card sprite sheets here.

Current wired asset:

- `arian-full-card-sheet.png`: 3 by 3 full-card sheet for Arian, the Still Colossus.
- `geert-full-card-sheet.png`: 3 by 3 full-card sheet for Geert, the Polarity Architect.
- `wouter-full-card-sheet.png`: 3 by 3 full-card sheet for Wouter, the Veilstalker.
- `noah-full-card-sheet.png`: 3 by 3 full-card sheet for Noah, Lord of Unwritten Chaos.

If the PNG is missing, the app falls back to the procedural prototype cards.
