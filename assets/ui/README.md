# Aetherfall UI Assets

Generated fantasy UI sheets and cropped runtime assets.

- `battlefield-aetherfall.png`: main battle board background.
- `champion-portraits-sheet.png`: source sheet for the four champion portraits.
- `portraits/`: cropped portrait images used by setup and hero panels.
- `card-backs-sheet.png`: source sheet for champion deck backs.
- `card-backs/`: cropped deck backs used by hidden hands and deck piles.
- `spell-effects-sheet.png`: source sheet for battle VFX.
- `effects/`: cropped VFX cells used by combat and spell animations.

The source sheets are kept so we can re-slice or improve the crops later without regenerating the art.
